Why we made Dungeon Hero

We are making this game Dungeon Hero  for assessment 3 in Certificate II in Game Programming (JavaScript).  This was our first game  design as a group.  We decided to make a dungeon crawler game because we had learnt how to code in tiles and it  uses all the concepts we have learnt so far during the course.
